# Visual Rules

Figma rules override legacy source styles.

| Role | Value |
|---|---|
| Primary | `#0080FF` |
| Success | `#2DA641` |
| Warning | `#FF8A00` |
| Danger | `#FF0000` |
| Page background | `#F7F8F9` |
| Table background | `#F8F8F8` |
| Card | `#FFFFFF` |
| Borders | `#DCDFE6`, `#E4E7ED`, `#EBEEF5`, `#F2F6FC` |

- Use PingFang SC: section title 16/24 semibold, body 14/22, secondary 13/21, auxiliary 12/20.
- Use black 85%, 65%, 45%, and 25% for primary, regular, secondary, and placeholder text.
- Use a `#F7F8F9` canvas, 24px content padding, 12–16px region gaps, white cards, and 4px radius.
- Use shadows only for overlays/floating elevation. Use state colors semantically and always pair risk/status color with text.
- Do not use `#409EFF`, oversized radii, gradients, glass effects, decorative illustrations, or excessive shadows.
