# 示例

- “生成一个科技金融潜客筛选页，支持行业、地区、科技等级筛选，展示企业卡、推荐等级和分页。” → `list`：筛选、已选条件、结果统计、企业卡、分页与状态。
- “生成一个企业风险画像页，首屏展示企业状态、信用等级、风险摘要和基础工商信息，下面展示风险事件列表。” → `enterprise-detail`：企业头部、等级标签、两列字段、指标和事件列表。
- “生成团队风险运营看板，按时间筛选，展示监控企业数、风险趋势、风险等级分布和最新动态。” → `dashboard`：时间筛选、最多四项指标、分析区、固定右栏动态。
- “生成产品分类管理页，左侧树形分类，右侧显示分类详情与别名列表，并提供新增和编辑入口。” → `master-detail`：搜索、32px 树节点、详情分组、附属表格、480px 表单弹窗。
